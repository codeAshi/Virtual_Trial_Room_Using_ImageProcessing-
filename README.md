# 👕 Virtual Trial Room

> 🚧 **Work in Progress**
> Under Construction

A full-stack web application that enables users to virtually try on clothes using computer vision and image processing techniques.

### 🚀 Highlights
- Real-time virtual clothing overlay
- Webcam and image upload support
- Body landmark detection using OpenCV & MediaPipe
- Spring Boot REST APIs
- React-based responsive UI

### 🛠️ Tech Stack
Java • Spring Boot • React.js • MySQL • OpenCV • MediaPipe

### 🎯 Goal
To improve the online shopping experience by allowing customers to visualize clothing before purchasing.
